前言

最后的结果就是期望，根据后验概率密度函数$f_k^+(x)$计算期望，得到结果

缺点：无穷积分计算量太大

改进：

主要代表是KF卡尔曼滤波和PF粒子滤波，下面将主要介绍。

卡尔曼滤波

主要思想

卡尔曼滤波的所有分布都是正态分布，递推计算各种期望，方差即可

初始条件：已知初始x0，所有观测{y}以及Q和R协方差矩阵，假设f为线性函数，可以是多项式

矩阵的计算公式（一般的应用都是矩阵形式的），具体推导思路：用一维的推导，将正态函数的概率密度函数代入贝叶斯滤波的公式中进行计算，得到新的分布函数（还是一个正态分布），摘取其期望和方差即可。

模型1简单的一阶线性函数

首先生成原始信号

t=0.1:0.01:1;
L=length(t);
%生成真实信号x，以及观测y
%初始化
x=zeros(1,L);
y=x;
%生成信号，x=t^2
for i=1:L
    x(i)=t(i)^2;
    y(i)=x(i)+normrnd(0,0.1);
end
plot(t,x,t,y)

写观测方程和预测方程，观测方程可以根据这个图像进行估算。（Q和R的选择：调参数）

%% 开始滤波
% 观测方程好写 yk=xk+r r~n(0,1)

%最粗糙的
%X(K)=X(K-1)+Q
%Y(K)=X(K)+R
%猜Q~N(0,1)
F1=1;
H1=1;
Q1=1;
R1=1;

Xplus1=zeros(1,L);
%设置一个初值，假设Xplus(1)~N(0.01,0.01^2)初值比较准
%X就是均值，P就是方差（协方差）
Xplus1(1)=0.01;%设置初值
Pplus1=0.01^2;%设置初值方差
for i=2:L
    % 预测步
    Xminus1=F1*Xplus1(i-1);
    Pminus1=F1*Pplus1*F1'+Q1;
    %更新步
    K1=(Pminus1*H1')*inv(H1*Pminus1*H1'+R1);
    Xplus1(i)=Xminus1+K1*(y(i)-H1*Xminus1);
    Pplus1=(1-K1*H1)*Pminus1;
end

plot(t,x,'r',t,y,'g',t,Xplus1,'b','LineWidth',1)

结果：可以看出滤波结果虽然有所减少，但是差距还是不小的

  • 无法回避的问题，Q和R的选择

模型2 多项式函数，多项式展开

%Model 2
%X(K)=X(K-1)+X'(K-1)*dt+X''(K-1)*dt^2*(1/2!)+Q2
%Y(K)=X(K)+R R~N(0,1)
%此时状态变量X=[X(K) X'(K) X''(K)]T 列向量
%Y(K)=H*X+R H=[1 0 0] (行向量)
%预测方程
%X(K)=X(K-1)+X'(K-1)*dt+X''(K-1)*dt^2*(1/2!)+Q2
%X'(K)=0*X(K-1)+X'(K-1)+X''(K-1)*dt+Q3
%X''(K)=0*X(K-1)+0*X'(K-1)+X''(K-1)+Q4  这个就是原来Model 1
%的预测方程，实际上就是猜是一个多项式，求几阶导也会平滑一点
%X''(K)=X''(K-1)+Q4 正是描述平缓的随机过程，这种建模相对精细一些，应用范围较广

dt=t(2)-t(1);
F2=[1 dt dt^2*0.5;
    0 1 dt;
    0 0 1];
H2=[1 0 0];%X中只有第一个有用
Q2=[1 0 0;
    0 0.01 0;
    0 0 0.0001];%协方差矩阵，代表了噪声之间的相关性，对角线元素是Q2,Q3,Q4
R2=20;
%%% 设置初值
Xplus2=zeros(3,L);
Xplus2(1,1)=0.1^2;%初值，只有第一个有用嘛
Xplus2(2,1)=0;
Xplus2(3,1)=0;
Pplus2=[0.01,0,0;
        0,0.01,0;
        0,0,0.0001];%初值
for i=2:L
    Xminus2=F2*Xplus2(:,i-1);
    Pminus2=F2*Pplus2*F2'+Q2;
    
    K2=(Pminus2*H2')*inv(H2*Pminus2*H2'+R2);
    Xplus2(:,i)=Xminus2+K2*(y(i)-H2*Xminus2);
    Pplus2=(eye(3)-K2*H2)*Pminus2;
end
plot(t,x,'r',t,y,'g',t,Xplus1,'b',t,Xplus2(1,:),'k','LineWidth',1)

（黑色的线）可以看出继模型1有不错的修正

模型2+二维观测

t=0.1:0.01:1;
L=length(t);
%生成真实信号x，以及观测y
%首先初始化
x=zeros(1,L);
y=x;
y2=x;
%生成信号，设x=t^2
for i=1:L
	x(i)=t(i)^2;
	y(i)=x(i)+normrnd(0,0.1);%正态分布，参数为期望和标准差
	y2(i)=x(i)+normrnd(0,0.1);
end
%问题2，两个传感器，进行滤波
% Y1(K)=X(K)+R
% Y2(K)=X(K)+R
%H=[1 1]T (列向量) X=X(K)
%H=1   0     0     X=X(K)   X'(K)   X''(K)
%     1   0     0
F3=[1,	dt,	0.5*dt^2;0,	1,	dt;0,	0,	1];%%%此处要注意矩阵是否病态，若dt特别小，易导致矩阵病态或精度丢失
H3=[1,0,0;1,0,0];
Q3=[1,	0,	0;0,	0.01,	0;0,	0,	0.0001];
R3=[3,0;0,3];%%%%%一定要注意是协方差矩阵
%%%设置初值%%%%
Xplus3=zeros(3,L);
Xplus3(1,1)=0.1^2;
Xplus3(2,1)=0;
Xplus3(3,1)=0;
Pplus3=[0.01, 0,	0;0,	0.01,	0;0,	0,	0.0001];
for i=2:L
	%%%预测步%%%
	Xminus3=F3*Xplus3(:,i-1);
	Pminus3=F3*Pplus3*F3'+Q3;
	%%%更新步%%%%
	K3=(Pminus3*H3')*inv(H3*Pminus3*H3'+R3);
	Y=zeros(2,1);
	Y(1,1)= y(i);
	Y(2,1)=y2(i);
	Xplus3(:,i)=Xminus3+K3*(Y-H3*Xminus3);
	Pplus3=(eye(3)-K3*H3)*Pminus3;
end
plot(t,x,'r',t,y,'g',t,y2,'b',t,Xplus2(1,:),'k',t,Xplus3(1,:),'m','LineWidth',2);

可以看出都比两个小

实际上也只是试一下怎么融合两个滤波而已了

需要注意的点

  1. 协方差矩阵的应用，一般只用对角线，表示噪声间不相关，对角线元素就是方差
  2. 矩阵的列向量和行向量的转化

粒子滤波

完整算法

  1. 给定初值$X_0 \sim N(\mu,\sigma)$
  2. 生成$x_0^{(i)},w_0^{(i)}=\frac{1}{n}$
  3. 预测步，生成$x_1^{(i)}=f(x_0^{(i)})+v$，v \sim N(0,Q)$的随机数，一共n个
  4. 更新步，设观测值为$y_1$，生成$w_1^{(i)}=f_R[y-h(x_1^{i})]$
  5. 归一化，$w_1^{(i)}=\frac{w_1^{(i)}}{\sum w_1^{(i)}}$
  6. 此时得到新的粒子$x_1^{(i)}$，新的权重$w_1^{(i)}$
  7. 重采样
  8. 求期望和方差
  9. 如此递推

总结（基本思想）

  1. 有大数定律，暗示pdf可由带权重的粒子表示
  2. 粒子数量，粒子位置，粒子完全决定了cdf，也决定了pdf
  3. 预测步改变粒子位置
  4. 更新步更新粒子权重

重采样

是什么

只有少数粒子具有较高的权重，大量粒子权重较低

为什么

原因：

  • 粒子太多，归一化归没了
  • 权重更新方程，$e^{-x^2}$，很容易就拉开差距了

坏处：导致权重更新失效

怎么做

  1. 按概率进行复制与淘汰，权重高的复制多几次，保证粒子数量不变
  2. 全部权重赋值为$\frac{1}{n}$
  3. （具体看代码）

作用

  1. 有一定削弱粒子退化的能力
  2. 减慢pf的速度

代码

先假设一个预测方程生成真实情况，然后根据真实情况模拟观测情况

% 预测方程 x(i)=sin(x(i-1))+5*x(i-1)/(x(i-1)^2+1)+Q
% 观测方程 y(i)=x(i)^2+R
clear
%注意，滤波之前一定要写状态方程观测方程形式
%就是这种xk=f(xk-1)
%大多情况下都是x=f(t)这样的
%要将它转换成上面的递推式，利用数值分析的常微分方程数值解就可以了

% 生成100个信号
t=0.01:0.01:1;
x=zeros(1,100);
y=zeros(1,100);
% t纯粹用来画图，由于直接给了xk与xk-1的关系，因此不需要用到dt
%给初值
x(1)=0.1;
y(1)=0.01^2;
%生成真实数据与观测数据
for i=2:100
    x(i)=sin(x(i-1))+5*x(i-1)/(x(i-1)^2+1);
    y(i)=x(i)^2+normrnd(0,1);
end
%figure(1)
%plot(t,x,t,y,'LineWidth',2);
% 例子滤波就是用y来预测x

%start
%设例子集合
n=100;
xold=zeros(1,n);
xplus=zeros(1,100);%用于放滤波值，后验概率的期望
xnew=zeros(1,n);
w2=zeros(1,n);
%设置初值x0(i)，可以直接在正态分布采样，如果对初值有信心，也可以让所有
%粒子相同，经过实验，可以发现跟初值其实没什么关系
for i=1:n
    xold(i)=0.1;
    w1(i)=1/n;
    w2(i)=1/n;
end
%核心代码

for i=2:100
    for j=1:n
        xold(j)=sin(xold(j))+5*xold(j)/(xold(j)^2+1)+normrnd(0,0.1);%Q
    end
    %更新步
    for j=1:n
        %w(j)=w(j)*fR(...)
        %fR=(2*pi*R)^(-0.5)*exp(/(y(i)-xold(j))^2/(2*R))
        w2(j)=exp(-(y(i)-xold(j)^2)^2/(2*0.1));%R
    end
    w2=w2/sum(w2);%w2是为了每一次重采样的
    
    %重采样
    %一般有限制例子退化程度N<1/sum(wi^2)
    c=zeros(1,n);
    c(1)=w2(1);
    for j=2:n
        c(j)=c(j-1)+w2(j);
    end
    for j=1:n
        a=unifrnd(0,1);
        for k=1:n
            if a<c(k)
                xnew(j)=xold(k);
                break; %一定要记住这个break
            end
        end
    end
    xold=xnew;
    for j=1:n
        w2(j)=1/n;
    end
    xplus(i)=sum(xnew)/n;
end
figure(2)
plot(t,x,t,y,t,xplus,'LineWidth',2);
legend('x','y','xplus')
